# Visual Review Notes

This directory stores screenshots used for manual visual review.

## Captures

- `desktop.png`: 1440 x 1200 viewport
- `mobile.png`: 390 x 900 viewport

The screenshots are generated by:

```bash
cd frontend
npm run dev
node tests/capture-screenshots.mjs
```

The capture script stubs backend responses, fills the app with representative
measurement and match data, waits for the top match to render, opens the
vs Target tab, enables overlap comparison mode, then captures the full page.

## Current Review Status

The trusted checks are:

- Playwright user-flow tests
- backend pytest coverage
- manual screenshot inspection of `desktop.png` and `mobile.png`
